# 变更日志

所有重要变更均记录在此文件中。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [未发布]

### 新增
- 全屏裁切编辑器：完整原图 + 半透明遮罩 + 白色裁切框，支持拖拽平移和双指缩放
- 全屏预览支持点击选中并交换子图位置
- 双击预览图进入全屏操作模式
- 13 种布局规则：AUTO / VERTICAL / HORIZONTAL / HERO_TOP / HERO_LEFT / HERO_RIGHT / GRID_2 / GRID_3 / GRID_4 / GRID_SQUARE / FIT_2 / FIT_3 / FIT_4
- 6 种输出比例：原始 / 1:1 / 4:3 / 3:4 / 16:9 / 9:16
- 单日图片增删：移除不想要的图片、从相册添加图片
- 一键重置当天编辑（恢复原始图片和裁切）
- CropOffset 模型支持 x/y 锚点 + 缩放倍率
- 布局评分算法根据图片实际宽高比智能选择最优布局
- 日期水印（白字 + 半透明黑底圆角背景 + 文字阴影）
- 可选删除原图功能（Android 11+ 系统确认弹窗保护）

### 修复
- 消除拼图黑边白边，改用 center-crop 缩放逻辑
